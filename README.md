## 👨‍💻‍基础配置信息
- 钱包地址中最少需要0.1个zeta
- 配置`data/config.py`的基础信息
- 输入邀请连接
- 配置`data/account.txt`文件，文件格式为`私钥::代理`，代理格式为：`用户名:密码@IP:端口`
